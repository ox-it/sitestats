Imprimir/Exportar Relatórios para arquivo

Depois de acessar um relatório existente (consulte Visualizar Relatórios item de ajuda para instrução de como acessar relatórios), relatórios podem ser imprimidos ou exportados para o arquivo.

Imprimir Relatórios

  • No topo da página do relatório, clique em Versão para impressão; uma versão pronta para impressão do relatório será apresentada.
  • Clicando em Enviar para impressão irá imprimir o relatório imediatamente.

Exportar um relatório para o arquivo

  • Na parte inferior do relatório, clique em Exportar ->; três botões adicionais substituirá o clicado.
  • Clicando em:
    • Exportar XLS exportará o relatório para um arquivo Microsoft Excel.
    • Exportar CSV exportará o relatório para um arquivo Comma Separated Values.
    • Exportar PDF exportará o relatório para um arquivo Portable Document Format.
